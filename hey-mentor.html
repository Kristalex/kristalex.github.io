Hey Mentor Mobile App

A mobile application for mentor-mentee communication.

Type

Short-term project

Role

UI/UX Designer, UX Researcher

Timeline

Jul 2018-Sep 2019

Hey Mentor is a non-profit organization that provides a mentorship program where college students help high-school graduates with applying to colleges.

I conducted user reasearch, designed a mobile application for mentor-mentee communication and helped with a website.

Jump to solution

User Research

I conducted user research as soon as I joined the team, even though the dev team had already started to code a mobile application. My goal was to see how the whole process looked at the moment for mentors and mentees, to define their pain points, and to understand how the app can help solve them.

Based on contextual interviews and quiestiionaries, I defined user needs and created user story maps to navigate the dev team through key scenarios for both mentors and mentees.

Mentor-mentee communication looked pretty simple from the first sight: connect, chat, share docs and links, and work with docs.

Findings

Communication. A mobile app indeed could help solve some problems in communication. Users mentioned that it was hard to navigate between different channels like fb, email, and messages. They had to be inventive in how to remind themselves to reply in time. They had no way of knowing if their message was read by another person if they sent it via email. All that could be fixed by a single app.

Sharing documents and links. The current version of the app was completely missing functionality for sharing documents and links which was the most important part of the mentorship.

Work with documents. Both mentors and mentees worked with documents on their computers and not on the phones. It meant that just a mobile application would break their flow.

Solution

I designed a mobile app that would improve communication between mentors and mentees. It would also serve as a useful tool for sharing documents and links. And the most important thing was that I designed a desktop version of the app to allow user to continue their journey when working with docs.

All communication with all mentees in one app

The app notifies when users need to reply

Users can see if their message was read and they can share files

Both mentors and mentees can see profiles of each other

Users can easily find shared links, files, and images
